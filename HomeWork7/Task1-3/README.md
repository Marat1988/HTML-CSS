# <b>Home work1-3</b>

<b>Задание 1.</b><br>

<p><b>Реализовать домашнюю страницу сайта-дневника.</b></p>
<p>На этой странице студент может просмотреть последние новости и список своих наград.</p>
<p>Блок новостей состоит из трех последних новостей и кнопки для загрузки предыдущих.</p>
<p>Одна новость состоит из:</p>
<ul>
<li>изображения (необязательно);</li>
<li>заголовка;</li>
<li>текста;</li>
<li>даты.</li>
</ul>
<p>Блок наград состоит из 9 последних наград и кнопки для загрузки предыдущих.</p>
<p>Одна награда состоит из:</p>
<ul>
<li>текстового описания;</li>
<li>даты выдачи;</li>
<li>количества наград;</li>
<li>иконки.</li>
</ul>
<p>На этой странице активным должен быть пункт меню Home.</p>
<p><b>При выполнении задания обязательно использовать Bootstrap.</b></p>
<p><b>Пример конечного результата:</b></p>

<b>Задание 2.</b><br>


<p><b>Реализовать страницу сайта-дневника с домашними заданиями.</b></p>
<p>На этой странице студент может посмотреть выданные ему задания, загрузить задание на проверку и узнать оценку.</p>
<p>Справа вверху отображаются условные цветовые обозначения.</p>
<p>Дальше домашние задания разделены на блоки по предметам. Сначала идет название предмета, потом – список заданий.</p>
<p>Одно задание состоит из:</p>
<ul>
<li>темы;</li>
<li>даты выдачи;</li>
<li>кнопки для загрузки/перезагрузки/удаления/скачивания задания;</li>
<li>оценки (если задание проверено).</li>
</ul>
<p>На этой странице активным должен быть пункт меню <b>Homework.</b></p>
<p>При выполнении задания обязательно использовать <b>Bootstrap</b></p>

<b>Задание 3.</b><br>
<p><b>Реализовать страницу сайта-дневника с профилем пользователя.</b></p>
<p>На этой странице студент может изменить информацию о себе или пароль для входа.</p>
<p>Страница состоит из трех блоков:</p>
<ul>
<li>загрузка изображения профиля;</li>
<li>форма с личными данными:
<ul>
<li>имя;</li>
<li>фамилия;</li>
<li>почта;</li>
<li>номер телефона;</li>
<li>адрес;</li>
<li>ссылка на социальную сеть.</li>
</ul>
</li>
<li>форма для изменения пароля.</li>
</ul>
<p>На этой странице активным должен быть пункт меню <b>Settings.</b></p>
<p><b>При выполнении задания обязательно использовать Bootstrap.</b></p>
